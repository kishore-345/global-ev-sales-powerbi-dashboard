# Global EV Sales Dashboard (2010–2024)
An interactive Power BI dashboard project analyzing Global EV Sales data (2010–2024) using data cleaning, DAX measures, Power Query, and advanced visualizations to uncover regional trends, powertrain insights, EV adoption growth, and time-series analytics.


## Project Overview

This project is an interactive Power BI dashboard developed using the IEA Global EV Dataset (2010–2024). The dashboard provides deep insights into electric vehicle adoption trends, regional performance, powertrain analysis, EV stock growth, and comparative analytics across different categories and years.

The project focuses on transforming raw EV data into meaningful business insights through data cleaning, DAX calculations, and interactive visual storytelling.

---

# Objectives

* Analyze global EV adoption trends from 2010 to 2024
* Compare EV growth across regions and vehicle categories
* Study powertrain performance and market dominance
* Understand EV stock, sales, and energy consumption trends
* Build an interactive and professional dashboard using Power BI
